> interpretation, validation, projection, and agent consumption. The conceptual
> contract lives in:

> End-to-end protocol demos, server processes, and integration tests belong in later
> implementation milestones after the adapter contract is ratified.
>
> See [docs/releases/alpha.md](../releases/alpha.md) for the current milestone
