# MCP server roots, resources, and tools

> **Superseded.** This milestone was replaced by the alpha architecture pivot
> ([#90](https://github.com/PathableAI-org/SpecAble/issues/90),
> [#95](https://github.com/PathableAI-org/SpecAble/issues/95)).
>
> MCP is now defined as an **adapter over the semantic product model**, not as a
> protocol-first server milestone. The conceptual contract lives in:
>
> **[MCP adapter](mcp-adapter.md)** — expose semantic product knowledge, validation
> findings, and derived artifacts to agents over the same wiki-backed model humans
> use; adapter-agnostic contract before protocol or implementation choices.
>
> Resource catalogs, tool enumeration, URI schemes, and server implementation belong
> in later milestones after the adapter contract is ratified.
>
> See [docs/releases/alpha.md](../releases/alpha.md) for the current milestone
> sequence.
